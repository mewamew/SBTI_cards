# Final SFX

这一层是当前仓库保留的最终运行时音效文件，游戏内直接按事件名取用即可。

## 选择结果

- `card_hover.wav`
  - 来源：`Playing Card Sounds / contact1.wav`
  - 原因：最短、最轻，不会把悬停做成现代 UI 铃声感。
- `card_select.ogg`
  - 来源：`54 Casino sound effects / card-place-2.ogg`
  - 原因：有明确的卡牌落位感，比通用 `tap` 更贴纸牌桌。
- `card_flip.wav`
  - 来源：`Playing Card Sounds / cut.wav`
  - 原因：干净的纸牌切换声，最接近翻牌瞬间。
- `ally_join.ogg`
  - 来源：`80 CC0 RPG SFX / spell_01.ogg`
  - 原因：带一点复古魔法感，但不电子。
- `ally_replace.ogg`
  - 来源：`54 Casino sound effects / card-shove-2.ogg`
  - 原因：有把卡推开再就位的动作感，适合替换伙伴。
- `enemy_hit.ogg`
  - 来源：`100 CC0 metal and wood SFX / wood_hit_04.ogg`
  - 原因：木质撞击更贴舞台感，不会太硬太现代。
- `ally_block.ogg`
  - 来源：`100 CC0 metal and wood SFX / metal_hit_02.ogg`
  - 原因：比敌人命中多一点金属感，更像护挡。
- `heal.ogg`
  - 来源：`80 CC0 RPG SFX / spell_02.ogg`
  - 原因：短促、轻微发亮，适合回血而不刺耳。
- `spy_reveal.wav`
  - 来源：`Card Game sounds / stagechange.wav`
  - 原因：短促的揭晓提示感强，适合 `FAKE` 暴露。
- `spotlight_reveal.ogg`
  - 来源：`54 Casino sound effects / cards-pack-open-1.ogg`
  - 原因：有开场和揭幕感，最接近聚光灯揭晓。
- `fate_reveal.ogg`
  - 来源：`100 CC0 metal and wood SFX / wood_hit_04.ogg`
  - 原因：更像木槌落下，适合放在翻牌完成后“命运落定”的裁决瞬间。
- `victory.wav`
  - 来源：`Win sound effect / Win sound.wav`
  - 原因：语义最明确，适合结果页胜利结算。
- `defeat.ogg`
  - 来源：`Lose Game Short Music Clip / losegamemusic.ogg`
  - 原因：虽然长度偏结果页短配乐，但当前语义最准确。
